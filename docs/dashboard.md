# Dashboard Global (Cockpit ComercialX)

## Endpoint principal
`GET /api/dashboard`

Devuelve un JSON con:

## KPIs del día
- ventas_dia
- transacciones_dia
- productos_vendidos_dia
- ticket_promedio_dia

## KPIs del mes
- ventas_mes
- transacciones_mes
- productos_vendidos_mes
- ticket_promedio_mes

## Ranking
- productos_mas_vendidos
- categorias_mas_vendidas

## Flujo de caja
- total_ingresos
- total_egresos
- balance

## Alertas
- alertas_no_leidas
- total_alertas
