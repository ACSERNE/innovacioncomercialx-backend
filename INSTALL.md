# Guía de instalación rápida

## Requisitos
- Node.js v22+
- Docker y Docker Compose
- PostgreSQL

## Pasos

1. Clonar el proyecto:
   git clone https://github.com/ACSERNE/innovacioncomercialx-backend.git

2. Instalar dependencias:
   cd backend
   npm install

3. Levantar con Docker:
   docker-compose up -d

4. Acceder a pgAdmin:
   http://localhost:8080
